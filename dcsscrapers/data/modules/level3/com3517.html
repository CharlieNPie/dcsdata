COM3517 Web Technologies

Summary

This course is concerned with getting acquainted with current Web technologies. We will read and discuss both teaching material and research papers. Topics will include:

  • Technologies for creating static and dynamic Web pages (e.g. HTML, JavaScript)  
  • Technologies for Web Databases (e.g. MySql)
  • Methods for searching and mining the WWW
  • Web 2.0 concepts and technologies
  • Client Server Architectures.
Session Spring 2018/19
Credits 10 credits
Assessment

Coursework [100%]
Lecturer(s) Dr. Vita Lanfranchi
Resources
  • MOLE
  • Unconfirmed practical marks when available
  • Exam Papers, past 2 years (where applicable)
Aims

This unit aims to make students aware of current web technologies, and to prepare for the future. The aim of the module is:

  • to describe the main web technologies
  • to teach how to critically analyse web solutions
  • to develop students' skills in designing and building solutions for the web
  • to create original solutions to emerging web trends and needs
Objectives

On completion of this course, students should:

  • be able to recognize/identify current web technologies.
  • be able to critically analyse web solutions and evaluate their appropriateness
  • be able to understand and explain the basics of web searching and web crawling.
  • be able to recognize and evaluate a number of state of the art web systems.
  • understand and explain emerging web standards
  • have a critical understanding of web 2.0 concepts
  • be able to devise original solutions to emerging web trends and needs
Content
  • Part I: Setting the Scene:
    • A history of the Internet and the Web
    • Present and Future Web
  • Part II: Web 1.0:
    • Building static and dynamic pages
      • HTML and XHTML
      • Javascript
    • Accessing Web databases (e.g. mySQL)
    • Client Server architectures
      • client server architectures with NodeJS
  • Part III:
    • Web search Engines, Bots, and Directories
    • Spidering the Web
    • Anatomy of a search engine
  • Part IV: Web 2.0 and beyond:
    • Web 2.0
      • The basics of the social Web
      • Major Web 2.0 web sites and services
      • Using Web 2.0 APIs and services (e.g. Twitter, Facebook, Google Maps etc.)
    • Capturing and using Collective Intelligence
  • Part V: Conclusions
Restrictions Available to students on the ITMB programmes only (COMU27 and COMU47). This module cannot be taken with COM3504.
Teaching Method Students learn the course matter through a mixture of lectures, tutorials and laboratories. Each week there will be 1 hour of lecture/tutorials and from the second week of semester, 2 hours of laboratory-based lesson.
Feedback Students will be given individual feedback on the first part of the assignment within three weeks of submission.
